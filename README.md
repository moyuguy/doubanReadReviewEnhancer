## 项目简介
豆瓣读书书评增强脚本是一款浏览器扩展/脚本，用于在豆瓣读书页面上加载来自Goodreads、Amazon、微信读书的评分信息，帮助用户更全面地了解书籍评价。

## 功能
- 在豆瓣读书页面上显示Goodreads评分
- 在豆瓣读书页面上显示Amazon评分
- 在豆瓣读书页面上显示微信读书评分

## 使用方法
1. 安装浏览器扩展或用户脚本管理器（如Tampermonkey或Greasemonkey）。
2. 创建新脚本，复制粘贴即可。
3. 访问豆瓣读书页面，相关评分将自动加载并显示。

## 兼容性
- 支持主流浏览器（Chrome、Firefox、Edge 等）
- 需要安装用户脚本管理器才能使用
